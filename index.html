Projects

Food Ordering app

A fullstack food ordering platform built with React and Node.js that allows users to browse menus, place orders, and complete payments securely. State is managed with redux-tookit which handles global state for cart auth etc. Features include JWT-based authentication with role-based access (user/admin), Stripe payment integration, an admin dashboard for managing orders and menu items with image uploads, and frontend testing with Vitest and React Testing Library. The application is deployed with a React frontend on Netlify and an Express/MongoDB backend on Render.
